

1. module_install.bat 실행해서 필요한 모듈 설치

2. python main.py 또는 본인이 갖고 있는 파이참, VS code에서 main.py 실행

2. Refresh 버턴을 눌러 준다
   : scenario 폴더 안에 있는 scenarios.json 파이을 읽어와 위젯을 구성 할 것임.

4. 테스트할 Test_Scenario 및 테스트 진행 할 모델 (알고리즘) 선택한 함.

4. 오른쪽 하단의 Analyze를 눌러 주면 평가하면서 Score 값이 반영될 것임

5. 평가 완료 후 필요하면 save 버턴을 눌러 현재 평가된 내용에 대해서 모두 저장 가능 함. (format은 .json 형태로 저장 됨)
